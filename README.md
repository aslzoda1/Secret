# 🧙‍♂️ Kotlin Magic Box: From Encryption to Life Simulation

Ushbu repozitoriya Kotlinda yozilgan ikkita qiziqarli va mutlaqo turlicha bo'lgan loyihani o'z ichiga oladi. Bu loyihalar Kotlinning ham algoritmlar, ham funksional dasturlash imkoniyatlarini namoyish etadi.

## 🚀 Loyihalar tarkibi

### 1. 🧬 Digital Ecosystem (Life Simulation)
Bu mashhur **Conway's Game of Life** algoritmining Kotlin-dagi minimalist talqini. 
* **Xususiyati:** Konsolda real vaqt rejimida hujayralarning tug'ilishi va o'lishi simulyatsiya qilinadi.
* **Texnologiyalar:** 2D Arrays, Game Loops, Thread Sleep.

### 2. 🔐 Secret Messenger (Caesar Cipher)
Xabarlarni shifrlash va deshifrlash uchun xavfsizlik vositasi.
* **Xususiyati:** Matnni kalit (shift) orqali shifrlaydi va qayta tiklaydi.
* **Texnologiyalar:** Functional Programming (`map`, `joinToString`), String Manipulation.
